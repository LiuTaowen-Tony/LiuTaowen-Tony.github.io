笔记：常系数线性微分方程

这篇里面怎么求特解的部分是我总结的，书上没有。

思路

L:(R→R)→(R→R)\mathcal L : (\R \rightarrow \R) \rightarrow (\R \rightarrow \R)L:(R→R)→(R→R) 是一个函数的函数它的定义是

L[y]=∑i=0kαi(x)Dk[y]\mathcal L [y] = \sum^{k}_{i=0}\alpha_{i}(x)\mathcal D ^{k}[y] L[y]=i=0∑k​αi​(x)Dk[y]

并且这个 L:(R→R)→(R→R)\mathcal L : (\R \rightarrow \R) \rightarrow (\R \rightarrow \R)L:(R→R)→(R→R) 是线性的。i.e.

L[λ1y1+λ2y2+λ3y3]=λ1L[y1]+λ2L[y2]+λ3L[y3]\mathcal L [\lambda_{1} y_{1} + \lambda_{2} y_{2} + \lambda_{3} y_{3}] = \lambda_{1} \mathcal L [y_{1}] + \lambda_{2}\mathcal L [y_{2}] + \lambda_{3}\mathcal L [y_{3}] L[λ1​y1​+λ2​y2​+λ3​y3​]=λ1​L[y1​]+λ2​L[y2​]+λ3​L[y3​]

所以两个微分方程的解相加依然是一个解。

我们要解一个形如

L[y]=f(x)\mathcal L [y] = f(x) L[y]=f(x)

的方程。因为这个L\mathcal LL 是线性的，所以我们可以先求零空间，然后求一个特解。

先求零空间：

find S, S={y∈(R→R)∣L[y]=0}S = \{y \in (\R \to \R) | \mathcal L [y] = 0 \}S={y∈(R→R)∣L[y]=0}

我们只需要找 k 个 线性不相关 的 方程 组成基底，就能求出来零空间了，对于 k 阶线性微分方程。

这 k 个线性不相关的方程一般是试出来的。

我们先猜 eaxe^{ax}eax 是一个类，然后解特征方程。(我懒得抄一遍笔记了)

然后我们得到一些特殊的 a。每一个 a 都代表一个一个满足 L[y]=0\mathcal L [y] = 0L[y]=0 的解。如果有重复的，我们还可以用 xeaxxe^{ax}xeax。

对于 2 阶的我们可以用这些。

特征方程有两个不同的根。

y=C1er1x+C2er2xy=C_{1}e^{r1x} + C_{2}e^{r2x} y=C1​er1x+C2​er2x

相同的根

两个虚根，我们可以用 sin 和 cos 把虚数包起来，所以整个是实数。

y=eax(C1cos⁡βx+C2sin⁡βx)y = e^{ax}(C_1\cos \beta x + C_2\sin \beta x) y=eax(C1​cosβx+C2​sinβx)

其中r1=a+bi,r2=a−bir_{1} = a + bi, r_{2} = a - bir1​=a+bi,r2​=a−bi

如何求特解

对于 2 阶微分方程，如果含有 e^x 的，我们只需要设两个待定系数就行了。因为一共可以用待定系数列出三个方程，其中有一个一定是 0 = 0.（可以从常数易变法得知）。

我们怎么设 ansatz，如果出现双曲函数，需要检验是不是和通解重复了。如果出现 cos sin 不能写成指数形式，因为常数易变法可能积不出来。cos sin 出现了一个 设的 ansatz 就要两个同时出现。出现 e^x 要检验是不是和通解重复，前面的多项式要设的高两次（因为求导两次），但是只需要两个待定系数。
