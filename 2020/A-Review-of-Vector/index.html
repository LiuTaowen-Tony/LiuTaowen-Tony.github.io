A Review of Vector

In this post we will go over the basic operation toward vector.

What is a vector

In the reallife, we do not always only want to look at the magnitude of a value, but sometimes we also need to concern about the direction of a value. For example, a boat in a river.

We can use a group of numbers to denote a vector. For example (x, y) or (x, y, z).

Zero vector

Some vectors has “strange” behaviour.

We prescribe zero vectors, (a1, a2, a3, … , an) where a1 = a2 = a3 = … = an = 0 and n ∈N∗\in \N ^*∈N∗, are parallel to any vectors. And all the zero vectors are equal, meaning (0, 0) is treated as the same as (0, 0, 0).

The definition of zero vector is just for convenience. And it is intuitive. All the vectors pointing to no direction and has no magnitude are the same.

Vector addition

We can only add the vectors in the same dimension (except zero vector).

Vector addition is defined as follow:

Two distinct vectors (a1, a2, a3, … , an) and (b1, b2, b3, … , bn) are denoted by a and b.

a + b := (a1 + b1, a2 + b2, a3 + b3, … , an + bn)
