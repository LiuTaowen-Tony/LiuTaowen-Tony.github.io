Matrix

Having the prerequisite knowledge of system of linear equations and vector, we can look at matrices.

I will first briefly tell you what a matrix looks like, and how is it work. Then there will be several examples to show why matrices are useful.

What is a matrix

(This part will be a little bit boring, because it is all definition.)

Matrix is a table of number.

Why are matrix vector multiplication helpful?

Maybe we can look at a example in computer graphics.

The matrix [2001]\begin{bmatrix}2&0\\0&1\end{bmatrix}[20​01​] would stretch a vector in x-axis by 2. I want to make an animation that a circle turns to a ellipse, then I can multiply the position of every pixel by this matrix.

That is a oversimplified occasion. In reality, we can use matrices to represent more complicated transformations, like rotation, shear, enlarge… And you can apply these transformations to higher dimensions.
