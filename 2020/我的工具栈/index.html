我的工具栈

这里放的是我手上现成学会的工具和目前要学的工具，帮助我理清在哪个方面我还缺趁手的工具。

Python

会了基本操作

numpy

矩阵运算，很多包的基础，高效计算必备

matplotlib

简单画画图，必备！貌似更高级的还有plotly，目前matplotlib和电脑自带的画图软件就够用了。我觉得我得自己写个什么速查表，这里命令老是忘，或者写个脚本辅助我画图。

sympy

符号运算！看心情选这个或者mathmatica，倾向于以后转到python上，因为python算矩阵方便一点

jupyter

前两天刚开始用的交互式环境，做数学很方便，和几个python包配合起来用起来跟mathematica差不多。难能可贵的是他有好看的符号输出，而且可以导出pdf和html。

以后还得学

scipy

科学运算包，数值运算很快，很重要，大概会了一些基本操作。以后还得看。

sklearn

机器学习轮子，只要调用就能得到很多好的东西。

django

搭网站的框架，以后博客迁到自己的服务器上都得靠这个。

还没学的

scrapy

爬虫，不了解，不解释

PIL pillow

图像处理

pandas

貌似是个数据处理库，感觉很有名

opencv

图像处理，计算机视觉用的很多

Javascript(啥也没学)

目前学了js的语法

以后还得学 node.js vue electron ionic
