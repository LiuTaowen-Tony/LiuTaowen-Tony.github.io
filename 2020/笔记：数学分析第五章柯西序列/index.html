给出概念：ϵ\epsilonϵ 稳定 和 最终 ϵ\epsilonϵ 稳定。最终ϵ\epsilonϵ 稳定是为了解决对数列开头的数字太敏感的问题。

有理数的柯西序列。

等价的有理柯西序列通过最终 ϵ\epsilonϵ 接近定义。

2 1.1 1.01 1.001 1.0001 …

等价的有理柯西序列。

之后我们需要证明形式极限可以加减乘除，需要注意我们还要证（1. 运算之后还是有理柯西序列 2. 等价的柯西序列替换之后的结果一样 LIM a = LIM b -> LIM a + LIM c = LIM b + LIM c)

对于除法，我们需要引入远离0的柯西序列。证明所有不等于0的有理柯西序列可以表示为远离0的柯西序列。

**注意！**不远离0的柯西序列可能整个都是正的：an=1/na_n = 1/nan​=1/n，但它的形式极限为0。

之后证远离0的柯西序列的倒数大都是柯西序列，并且定义明确。

最小上界性质

最小上界唯一

实数的最小上界存在性：如果E是实数集的一个非空子集，且有一个上界M，那么一定有一个最小上界。

因为最小上界唯一，所以我们证至少存在一个最小上界。

我们利用两个分数的柯西序列从上面和下面逼近这个最小上界。

我们希望做两个有理数列

K11 K22 K33 K44 ...\frac{K_1}{1}\ \frac{K_2}{2}\ \frac{K_3}{3}\ \frac{K_4}{4}\ ...1K1​​ 2K2​​ 3K3​​ 4K4​​ ...

L11 L22 L33 L44 ...\frac{L_1}{1}\ \frac{L_2}{2}\ \frac{L_3}{3}\ \frac{L_4}{4}\ ...1L1​​ 2L2​​ 3L3​​ 4L4​​ ...

其中K 序列都是这个数列的上界，L 序列都不是这个数列的上界。

为什么要用K序列和L序列：我们不好直接取M序列，我们一开始只知道有一个上界，通过这个上界根据阿基米德性质构造K和L，这样我们就收缩到一个有限集里面，在有限集里面我们就能挑选符合我们条件的M了。

根据阿基米德性质，我们要找一个M序列

M11 M22 M33 M44 ...\frac{M_1}{1}\ \frac{M_2}{2}\ \frac{M_3}{3}\ \frac{M_4}{4}\ ...1M1​​ 2M2​​ 3M3​​ 4M4​​ ...

其中 Ln<Mn≤KnL_n < M_n \le K_nLn​<Mn​≤Kn​ ，序列中的每个元素都是上界并且

M1−11 M2−12 M3−13 M4−14 ...\frac{M_1 - 1}{1}\ \frac{M_2 - 1}{2}\ \frac{M_3 - 1}{3}\ \frac{M_4 - 1}{4}\ ...1M1​−1​ 2M2​−1​ 3M3​−1​ 4M4​−1​ ...

都不是上界。

之后我们证明这个M序列是柯西序列。

我们需要借助上下两个序列来证。

mnn>mn′−1n′mnn>mn′n′−1n′mnn−mn′n′>−1n′mn−1n<mn′n′mnn−1n<mn′n′mnn−mn′n′<1n∣mnn−mn′n′∣<∣1n∣\frac{m_n}{n} > \frac{m_{n'}-1}{n'}\\ \frac{m_n}{n} > \frac{m_{n'}}{n'}-\frac{1}{n'}\\ \frac{m_n}{n} - \frac{m_{n'}}{n'} > -\frac{1}{n'}\\ \frac{m_n - 1}{n} < \frac{m_{n'}}{n'}\\ \frac{m_n}{n} -\frac{1}{n}< \frac{m_{n'}}{n'}\\ \frac{m_n}{n} -\frac{m_{n'}}{n'} < \frac{1}{n}\\ |\frac{m_n}{n} -\frac{m_{n'}}{n'}| < |\frac{1}{n}|\\ nmn​​>n′mn′​−1​nmn​​>n′mn′​​−n′1​nmn​​−n′mn′​​>−n′1​nmn​−1​<n′mn′​​nmn​​−n1​<n′mn′​​nmn​​−n′mn′​​<n1​∣nmn​​−n′mn′​​∣<∣n1​∣

这样这个序列是有理柯西序列，所以是一个实数。

之后我们证它是一个上界，并且所有其他上界都大于等于这个上界。

因为M序列中每一个元素都大于等于这个集合的任意元素，所以极限也是上界。

因为M序列和M-1序列是极限相同，并且所有上界都大于等于M-1序列的任意元素，所以这个序列是最小上界。
